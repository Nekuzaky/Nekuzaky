## 🧑‍💻 About Me

Hey, I'm **Nekuzaky** — open-source developer and founder of **[Altcore](https://www.nekuzaky.com/)**.

- 🎮 Unity & C# — game tools, game jams, modular systems
- 🌐 JavaScript / TypeScript — web apps, Discord bots, online tools
- 🪓 Minecraft Mods (Java / Fabric) — gameplay mechanics & mod systems
- 🚀 Founder of [Altcore](https://www.nekuzaky.com/) — fast tools for creators, devs & gamers
